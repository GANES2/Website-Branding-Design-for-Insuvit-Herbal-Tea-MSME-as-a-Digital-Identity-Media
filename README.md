# Website Branding Design for Insuvit Herbal Tea MSME as a Digital Identity Media

This repository contains the high-fidelity web prototype for Insuvit Herbal Tea, serving as a digital identity and centralized information hub for the MSME.

## Technologies Used
- HTML5
- CSS3 (Vanilla)
- JavaScript (Vanilla)

## Features
- Fully responsive design
- No medical claims (academic/prototype safe)
- Optimized Call-to-Action to Shopee
- Smooth scroll & micro-interactions

*Built for academic research purposes.*
